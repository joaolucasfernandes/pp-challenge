Bugs e sugestões de melhoria:

(Post - Criar Usuário)
1. Não enviar valores relacionados aos campos no Json, causa um erro 500. Tratar seria legal.
2. Quando você coloca um número no campo nome no payload, a API aceita. Talvez uma validação aceitando só Strings valeria a pena.
3. Status codes dos requests de validação de campos está como 200, dando o código na resposta. Alterar para 400 facilitaria a vida dos clients
4. Status code dos requests com autenticação invalida retorna status code 200. Tratar para 401, pelo mesmo motivo acima.
5. Verificar com o time de negócios se o uso de um timezone diferente do Brasil não causaria problemas em algum nível.
